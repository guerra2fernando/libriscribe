# book-outliner

> **Category:** Standalone specialist
> **Source:** LibriScribe `outliner` + `scene_outliner` (adapted)
> **May produce prose:** Outline documents only
> **Available:** Any time — best used during stage 2

## Purpose

Produce a detailed chapter-by-chapter outline and, optionally, scene-level breakdowns within chapters. Ensures strong narrative or argumentative structure before writing begins.

## Inputs

- `01-init.md` (concept, audience, genre, length intent)
- Book length / chapter count target
- Any user notes on intended structure

## Process

1. Draft a high-level chapter list with one-sentence descriptions.
2. For each chapter: identify the primary purpose (advance plot, deepen character, deliver argument, establish world).
3. Mark chapters with content dependencies (chapter N requires chapter M to be written first).
4. Flag chapters safe for parallel drafting.
5. For chapters needing scene-level detail: break into 3–5 scene beats.
6. Verify pacing and story arc across the full outline.

## Output

- **Chapter list** — number, title, 1–3 sentence summary, word count target
- **Scene beats** (optional, per chapter) — action + purpose
- **Dependency annotations** — which chapters must precede others
- **Parallel flags** — chapters safe to draft concurrently
- **Pacing notes** — structural observations (e.g., "tension drops in chapters 5–6, consider adding conflict beat")
