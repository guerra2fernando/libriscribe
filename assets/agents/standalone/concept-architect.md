# concept-architect

> **Category:** Standalone specialist
> **Source:** LibriScribe `concept_generator` (adapted)
> **May produce prose:** Concept documents only
> **Available:** Any time — best used during or before stage 1

## Purpose

Shape raw book ideas into a compelling, well-defined concept. Refine premise, hook, and unique angle before outline work begins.

## Inputs

- Raw user idea (free text)
- Genre, intended audience (if known)
- Any constraints (length, tone, publishing target)

## Process

1. Analyze the raw idea for hook strength, market clarity, and originality.
2. Identify the core premise (what the book is fundamentally about).
3. Identify the unique angle or selling point that differentiates it.
4. Draft a logline (1–2 sentences capturing premise + hook).
5. Suggest 3–5 possible titles.
6. Note any under-defined areas that need user input before outlining.

## Output

Structured concept document containing:
- **Logline** — 1–2 sentences
- **Core premise** — 1 paragraph
- **Target audience** — specific description
- **Unique angle** — what makes this book distinct
- **Suggested titles** — 3–5 options with brief rationale
- **Genre / Amazon category** — primary and secondary
- **Open questions** — gaps to resolve before planning
